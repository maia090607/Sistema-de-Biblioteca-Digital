# Sistema de Biblioteca Digital - Backend ORM

# INSTALACION 
1. Clonar el repositorio
    git clone https://github.com/maia090607/Sistema-de-Biblioteca-Digital.git 
    
2. Entrar a la carpeta
    cd sistema-biblioteca-digital

3. Instalar las dependencias del proyecto
    npm install

4. Ejecutar el sistema y la suite de pruebas
    node app.js

# EJECUTAR ARCHIVO biblioteca.sqlite
1. Ir a extenciones y descargar SQLite Viewer" (de Florian Klampfer)

2. Ir al explorador y dar clic derecho sobre el archivo biblioteca.sqlite y seleccioanr la opcion abrir con... y seleccioanr SQLite Viewer
