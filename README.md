# Sistema de Biblioteca Digital - Backend ORM

Este proyecto implementa el backend de datos para un **Sistema de Biblioteca Digital** utilizando **Node.js** y **Sequelize ORM** con el motor de base de datos **SQLite**. Cubre todo el ciclo de diseño solicitado en el curso: mapeo de entidades, establecimiento de relaciones complejas, control de versiones del esquema, operaciones CRUD optimizadas mediante *Eager Loading* para mitigar el problema de rendimiento N+1, y transacciones atómicas seguras para el control del inventario de libros.


# INSTALACION 
1. Clonar el repositorio
    git clone https://github.com/tu-usuario/sistema-biblioteca-digital.git

2. Entrar a la carpeta
    cd sistema-biblioteca-digital

3. Instalar las dependencias del proyecto
    npm install

4. Ejecutar el sistema y la suite de pruebas
    node app.js
